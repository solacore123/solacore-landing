# 솔라코어 (Solacore) 태양광 설치 상담 랜딩 페이지

## 🌟 프로젝트 개요
(주)솔라코어의 태양광 발전 시스템 설치 상담을 위한 전문 랜딩 페이지입니다.

**핵심 메시지**: "노는 옥상 돈 벌게 해드려요 - 연 3,000만원 이상 수익 발생"

## 🎯 **페이지 구조 재설계 (전환율 최적화)**

**기존 문제점**: 랜딩 페이지 구조가 정보 나열 중심으로 전환 동선이 비효율적

**최적화 전략**: 사용자 심리 단계에 맞춘 전환 퍼널 설계
1. **관심 유발** → ROI 계산기로 개인화된 가치 제안
2. **신뢰 구축** → 실제 데이터 기반 Before/After 증거
3. **행동 유도** → 낮은 장벽의 사진 업로드 CTA
4. **정보 제공** → 이후 상세 정보 및 신뢰 요소

**예상 전환율 개선**: 기존 2~3% → 6~8% (2~3배 증가)

## 📋 완성된 기능 (최적화된 순서)

### ✅ 1페이지: 히어로 섹션
- 실제 태양광 설치 현장 배경 이미지
- 핵심 메시지: "노는 옥상 돈 벌게 해드려요"
- 가치 제안: "연 3,000만원 이상 수익 발생"
- **CTA 버튼 최적화**: 
  - 주요: "내 건물 ROI 확인" (개인화, 상호작용 유도)
  - 부차: "사진으로 빠른 상담" (낮은 장벽)
- 신뢰 지표: 500+ 시공 실적, ISO 9001 인증, 10년 품질보증, 정부지원사업 협력

### ✅ 2페이지: ROI 투자 회수 계산기 ⭐ (이동: 5페이지 → 2페이지)
**"내 건물은 몇 년이면 투자비를 회수할까요?"**

**🎯 전환율 최적화 핵심 전략**:
- **위치 최적화**: Hero 바로 다음 배치 → 즉시 가치 체감
- **개인화**: 사용자가 직접 입력하여 나의 ROI 확인
- **상호작용**: 슬라이더 조작으로 능동적 참여 유도
- **신뢰감**: 범위 표시(7~9년)로 현실성 있는 정보 제공

**인터랙티브 계산기 기능**:
- 슬라이더로 월 평균 전기요금 입력 (50만원 ~ 1,000만원)
- 실시간 자동 계산:
  - **예상 투자 회수 기간** (범위 표시: 예: 7~9년)
  - 예상 설치 용량 (kW)
  - 연간 예상 수익
  - 이후 15년간 순수익 예상
- 시각적 타임라인 바 (투자 회수 vs 순수익 발생 기간)

**ROI 범위 표시** (현실적이고 신뢰감 있는 표현):
- 50~100만원: **11~13년**
- 100~200만원: **9~11년**
- 200~350만원: **7~9년**
- 350~550만원: **6~8년**
- 550~750만원: **5~7년**
- 750만원+: **5~6년**

**계산 로직** (현실 반영):
- 용량 추정: 티어별 공식 (작은 건물 0.5, 큰 건물 0.35 계수)
- 투자 비용: 규모별 할인 (30kW 이하 200만원/kW → 300kW+ 150만원/kW)
- 연간 수익: 고정 80% 절감율
- ROI 기간: 월 전기요금 구간별 범위 (6개 브래킷)

**CTA 버튼**: "정확한 견적 받기" (바로 상담 섹션으로 이동)

### ✅ 3페이지: 설치 전·후 실제 전기요금 비교 ⭐ (신규 독립 섹션)
**"실제 청구서로 증명합니다"**

**🎯 전환율 최적화 핵심 전략**:
- **사회적 증거**: 추상적 수치가 아닌 실제 청구서 스타일
- **시각적 임팩트**: 빨강(Before) → 초록(After) 색상 대비
- **구체적 수치**: "84% 절감, 연 2,880만원" 명확한 혜택
- **감정 자극**: 😞 → 😊 이모티콘으로 감정 이입

**Before/After 데이터** (공장 건물 100kW 기준):
- **설치 전**: 월 285만원 (연 3,420만원) 😞
- **설치 후**: 월 45만원 (연 540만원) 😊
- **절감 효과**: 84% 절감, 연간 2,880만원 수익 발생 ✅

**디자인 특징**:
- 실제 청구서 스타일 카드 (헤더, 금액, 날짜, 상세)
- 애니메이션 화살표 (좌→우 슬라이드 반복)
- 초록 배지 "84% 절감!" 강조
- 카드 순차 페이드인 애니메이션

**CTA 버튼**: "우리 건물도 가능한지 사진으로 확인하기" → 사진 업로드 섹션

**디자인 특징**:
