## 📋 설정 단계 (초간단!)

### 1단계: Google Sheets 만들기 (1분)

1. https://sheets.google.com 접속
2. **새 스프레드시트** 만들기
3. 이름: "솔라코어 상담 데이터"
4. 시트 2개 만들기:
   - **"상담신청"** (기본 시트 이름 변경)
   - **"사진업로드"** (+ 버튼으로 추가)

**상담신청 시트 첫 줄**:
```
이름 | 연락처 | 건물유형 | 면적(평) | 월전기요금(만원) | 문의내용 | 제출시간
```

**사진업로드 시트 첫 줄**:
```
연락처 | 파일이름 | 파일크기(MB) | 제출시간
```

---

### 2단계: Apps Script 설정 (2분)

1. 스프레드시트 상단 → **확장 프로그램** → **Apps Script**
2. 기본 코드 전체 삭제
3. 📋 `GOOGLE_SHEETS_SETUP.md` 파일 열기
4. **2-2. 코드 붙여넣기** 섹션의 코드 **전체 복사**
5. Apps Script 에디터에 **붙여넣기**
6. **5번째 줄 수정**:
   ```javascript
   var ADMIN_EMAIL = "jimy9902@naver.com";  // ← 이메일 확인
   ```
7. **Ctrl + S** (저장)

---

### 3단계: 배포하기 (1분)

1. 상단 **배포** 버튼 → **새 배포**
2. ⚙️ 아이콘 → **웹 앱** 선택
3. 설정:
   - **다음 계정으로 실행**: 나
   - **액세스 권한**: **모든 사용자** ⚠️ 중요!
4. **배포** 클릭
5. **권한 검토** → 계정 선택
6. "확인하지 않은 앱" 경고 → **고급** → **이동** → **허용**
7. ✅ **웹 앱 URL 복사** (예: `https://script.google.com/macros/s/AKfyc...`)

---

### 4단계: 웹사이트 코드 수정 (1분)

1. 프로젝트에서 `js/main.js` 파일 열기
2. **7번째 줄** 찾기:
   ```javascript
   const GOOGLE_SCRIPT_URL = 'YOUR_GOOGLE_SCRIPT_URL_HERE';
   ```
3. 3단계에서 복사한 URL로 교체:
   ```javascript
   const GOOGLE_SCRIPT_URL = 'https://script.google.com/macros/s/AKfyc.../exec';
   ```
4. **저장**

---

### 5단계: 테스트! (1분)

#### Apps Script 테스트:
1. Apps Script 에디터 상단 함수: **testContactForm** 선택
2. **실행** (▶️) 버튼 클릭
3. Google Sheets → "상담신청" 시트 확인 → 테스트 데이터 있음 ✅
4. 이메일 확인 → 알림 수신 ✅

#### 웹사이트 테스트:
1. 웹사이트 열기
2. 상담 신청 폼 제출
3. Google Sheets 확인 → 데이터 저장됨 ✅
4. 이메일 수신 확인 ✅

## 🚨 문제 해결

### "권한 오류" 발생
→ 배포 시 **"모든 사용자"** 선택했는지 확인

### 데이터가 저장 안됨
→ Google Sheets 시트 이름 확인:
- "상담신청" (정확히)
- "사진업로드" (정확히)

### 이메일이 안옴
→ `ADMIN_EMAIL` 확인  
→ 스팸함 확인  
→ Gmail 사용 권장

### CORS 오류
→ 정상입니다! `mode: 'no-cors'`로 해결됨  
→ 데이터는 정상 전송됨
